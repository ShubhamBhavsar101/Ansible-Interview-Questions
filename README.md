
# Ansible Interview Questions

## 1. What is Ansible?

Ansible is an open-source configuration management and automation tool used to provision infrastructure, deploy applications, and manage systems.

## 2. Why is Ansible Agentless?

Ansible is agentless because it uses SSH (or WinRM for Windows) to communicate with nodes, so no agent needs to be installed on managed hosts.

## 3. What is a Playbook in Ansible?

A playbook is a YAML file that defines automation tasks, configurations, and workflows to be executed on target hosts.

## 4. What is a Play?

A play maps a set of tasks to a specific group of hosts and defines how those tasks should be executed.

## 5. What is a Variable?

A variable is used to store values that can be reused across playbooks, roles, and tasks to make automation dynamic and flexible.

## 6. What is Jinja2 Templating?

Jinja2 is a templating engine used in Ansible to dynamically generate configuration files using variables and expressions.
